Sociocracy Core Ideas

  • Make sure everyone is heard.
  • All objections must be addressed.
  • No voting to override minority voices.
  • Delegate Responsibilities

Sociocracy: Circles

  • Hub
  • Purpose specific subcircles
  • Between 5-8 people ideal

Sociocracy: Delegation

  • Delegate control with work
  • Delegate to subcircle or individual with role

Sociocracy: Core Roles

  • Leader
  • Facilitator
  • Secretary
  • Delegate if sub-circle
  • All roles with timeframe

Sociocracy: Talking in Rounds

  • Check-in round
  • Get consent for agenda
  • If impractacle then circle is too big!

Sociocracy: Consent

  • Not the same as consent in sex & kink!
  • Preference vs Toleration
  • Good enough for now, safe enough to try.

Sociocracy: Transparency

  • Default to transparency
  • Except for handling conduct issues

Sociocracy: Breaking a Jam

  • Randomly temporarily kick members from circle.

Conduct Issue

  • Handle with for-purpose circle
  • Individual involved can't be in issue circle
  • Have documented procedure and standard handling

Personal Conflict

  • Hopefully addressed as conduct issue
  • Mediation?
  • Last resort, the Ultimatum: them or me?
  • Unless all consent to kicking out the accused
